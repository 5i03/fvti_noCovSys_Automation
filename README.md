# fvti_nCov_Auto_Reporter
福州职业技术学院健康管理系统自动化脚本
The automation script of the "JianKangLiYuan" applet of Fuzhou Polytechnic(fvti)
支持功能如下:
1. 自动更新令牌,无需有人值守
2. 支持辅导员自动批准当日往返出入校申请
3. 支持统计未填写日报的人名(其他信息需教师账户支持)
4. 支持学生自动化填写健康日报
5. 支持5i03推送服务,无需登入服务器,即可知道执行状态
